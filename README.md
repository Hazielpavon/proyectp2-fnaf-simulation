# Proyecto P2 - FNAF Simulation

Este repositorio contiene un proyecto desarrollado en Java como parte de mi proceso de aprendizaje en la universidad.

El objetivo principal de mantener este repositorio publico es demostrar mi avance en programacion, especialmente en temas relacionados con Programacion Orientada a Objetos, interfaces graficas, manejo de clases, objetos, eventos y estructura general de un proyecto en Java.

## Proposito del repositorio

Este proyecto forma parte de mi crecimiento academico y personal como estudiante de Ingenieria en Sistemas. A traves de este repositorio busco evidenciar la evolucion de mis conocimientos, la practica constante y la mejora progresiva en el desarrollo de software.

## Tecnologias utilizadas

* Java
* NetBeans
* Swing
* Programacion Orientada a Objetos

## Temas aplicados

* Clases y objetos
* Herencia
* Interfaces graficas
* Eventos
* Manejo de imagenes
* Estructura de proyectos Java
* Logica de simulacion

## Nota

Este repositorio se mantiene publico con fines academicos y de portafolio. El codigo representa una etapa de aprendizaje, por lo que puede contener areas de mejora, pero tambien refleja el progreso adquirido durante la carrera universitaria.
